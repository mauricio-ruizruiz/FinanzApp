
# FinanzApp

FinanzApp es una aplicación de finanzas personales diseñada para ayudarte a monitorear tus ingresos y gastos de manera anual, permitiendo una gestión eficiente de tu economía.

## Características principales

- **Presupuestos mensuales:** Define y ajusta presupuestos para cada mes del año, facilitando el control de tus gastos.
- **Registro de ingresos y gastos:** Añade y categoriza cada movimiento financiero para llevar un historial detallado.
- **Balances y reportes:** Visualiza balances mensuales y anuales, analiza el comportamiento de tus finanzas y toma mejores decisiones.
- **Panel de resumen:** Consulta rápidamente el estado actual de tus finanzas, con gráficos y estadísticas.

## Objetivo

El objetivo de FinanzApp es ofrecer una herramienta intuitiva y completa para el seguimiento financiero personal, ayudando a los usuarios a alcanzar sus metas económicas mediante el control y análisis de sus ingresos y gastos.
